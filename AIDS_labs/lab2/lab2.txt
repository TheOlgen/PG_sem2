Sorting (labs)
POLSKI
Należy posortować zadany zbiór dużych liczb naturalnych, używając algorytmu podanego przez prowadzącego (możliwe że mergesort
zapytaj prowadzącego, -1024 pkt. wykorzystanie innego). 
Nie używaj wbudowanych funkcji sort(), qsort() oraz compare(), które rozwiązują częściowo lub całkowicie problem (-512 pkt.).
Input

W pierwszej linii liczba elementów zbioru N, w następnych N liniach liczby do posortowania.


Output

Posortowany ciąg wyrazów w kolejności niemalejącej, rozdzielony znakiem nowej linii.


Example

Input:
5
13
27
328
111
25

Output:
13
25
27
111
328


ENGLISH
Sort a sequence of large numbers using an algorithm given by the teacher (maybe mergesort but ask the teacher, -2048 pts.). 
You can find a pseudocode of the algorithm here.  Do not use sort(), qsort(), compare() etc. that solve partially or completely the problem (-512 pkt.).

Input

In the first line a number of integers N. In next N lines numbers to sort.


Output

The sorted sequence in non-decreasing order, separated by new lines.


Example

Input:
5
13
27
328
111
25

Output:
13
25
27
111
328
